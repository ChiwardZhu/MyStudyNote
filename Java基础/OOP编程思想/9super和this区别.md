## super和this的区别

#### super

1. super()：是调用父类的构造方法，这行代码必须写在子类构造方法的第一行。
2. super只能出现在子类的方法或者构造方法中
3. super和this不能同时调用构造方法
4. 子类所有的构造方法调用父类无参构造器super()；
5. 构造方法的第一句总是：super(..)来调用父类对应的构造方法



#### this

1. 代表的对象不同：

   this：这个方法的对象（每个方法只有一个）

   super：父类方法的对象

2. 前提：

   this：没有继承也可以试用

   super：只有在继承下才能试用

3. 构造方法：

   this（）;：调用本类的构造方法

   super（）；：调用父类的构造方法
