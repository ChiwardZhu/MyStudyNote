## 1.1数组的概念

**数组的定义：**数组是==相同类型==数据的==有序==集合

**数组的特点：**

1. 长度是确定的。一旦被创建，它的大小就是不可以改变的
2. 其元素的类型必须是相同类型
3. 数组类型可以是任何数据类型（基本类型和引用类型）
4. 数组变量属于引用类型，数组也是对象

* **注意：**

  数组变量属于引用类型，数组也是对象，数组中的每个元素相当于该对象的成员变量。

  数组本身就是对象，Java中对象是在堆中的，因此数组无论保存原始类型还是其他对象类型，数组对象本身存放在堆中
